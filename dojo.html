CSS Dojo

In each kata below, you will be instructed to change the styles applied to the kata such that the text of the kata matches its style. You may only make changes that conform to the rules set out in each individual kata. You should also leave the kata ID selectors alone so that your changes only affect the kata you're working on.

Kata 1

Remove the !important declaration to make this sentence italic. Then reorder the rule sets to change it back.

Kata 2

Modify a selector to make it so that this <span> has a dashed border but this <span> has a solid border.

Kata 3

Modify a selector to make it so that this <strong> is rendered in small caps while this <strong> is not.

Kata 4

Modify the selector so that only checked checkboxes have the grab cursor when you hover over them.

Kata 5

Modify the selector so that the TV Tropes icons seen in this paragraph are only applied to links to tvtropes.org. If you're having trouble, this MDN page. may help. When you have some free time and would like to amuse yourself, have fun reading about The Evil Overlord List, The Jargon File, or Growing the Beard. Maybe you need to check to see if TV Tropes is down? But beware, because TV Tropes will ruin your life.

Kata 6

Each sentence in this kata is bold and black with a white glow. However, I want each sentence to alternate the white and red glows, starting with red. Create a new rule set to do this.

Kata 7

If you are a fan of the Monkey Island games, you know that Guybrush Threepwood is afraid of porcelain. So to avoid traumatizing him, we will hide every instance of the word “porcelain” in this paragraph. There are two style rule sets for this kata in the style sheet, and you may only modify the selector for the first one, and you cannot reorder them. Make the porcelain go away!

Kata 8

Below there are four invisible playing cards. If you mouse over them, they flip over to display their faces. The backs (with the back class) are invisible because the transform property is being overridden by another style rule, causing it to be rotated the same direction as the front. Change the specificity of that selector so that the backs of the cards are visible.
